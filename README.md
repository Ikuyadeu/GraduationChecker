# GraduationChecker
ソフトウェア工学で作成した卒業研究に参加可能かどうかをチェックするシステム

単位数を入力することで参加可能判定、必要な単位数、不足分単位数を出力  

* 開発環境  
  言語：　Java7  
  eclipse: Luna 
  
* 実行方法 
  ```
  $ cd src 
  $ java GraduationChecker.java
  ```

* 追加予定機能  
1.学務情報システムからのCSV形式のデータ読み込み（.txt,.csv対応）  
2.出力データの保存(.txt,.csv形式)  
3.他学科・他学部への対応  
4.必修科目の出力
